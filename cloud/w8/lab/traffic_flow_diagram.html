Tóm tắt luồng đi của request (Flow Summary)

  1. Request: Client truy cập domain của ALB trên Internet qua cổng HTTP 80.
  2. ALB Routing: ALB nhận và chuyển tiếp request đến Target Group trên cổng 30080 của EC2.
  3. Host Listening: Tiến trình socat trên EC2 đón gói tin trên cổng 30080.
  4. K8s Forwarding: socat chuyển hướng gói tin vào IP LoadBalancer của Minikube.
  5. Pod Response: Kube-proxy định tuyến đến Pod disc-player, Nginx trả về assets và stream video.

Điểm nhấn bảo mật (Security Highlights)

  • Network Firewall: EC2 chỉ nhận traffic từ ALB Security Group trên cổng 30080. Không thể bypass để hack trực tiếp EC2.
  • ALB Hardening: ALB lọc và bỏ qua các header HTTP bị lỗi để chống tấn công HTTP Smuggling.
  • No Access Keys: Dữ liệu video tĩnh được tải từ S3 Bucket bằng IAM Instance Profile động, không lưu AWS credentials trên máy.
